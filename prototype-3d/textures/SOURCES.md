# Texture sources

All textures here are **CC0 (public domain)** from **ambientCG**
(https://ambientcg.com), free for commercial use with no attribution required.
Kept maps: Color (sRGB), NormalGL, Roughness — 1K JPG.

| Folder    | ambientCG asset | Used for            |
| --------- | --------------- | ------------------- |
| `wall/`   | Bricks097       | hall walls + dais   |
| `column/` | Marble016       | column shafts       |
| `dark/`   | Rock030         | throne, ceiling, column caps |

Other good CC0 sources for later: Poly Haven (textures + HDRIs), freestylized
(stylized, free for commercial use — don't re-host the raw packs).
